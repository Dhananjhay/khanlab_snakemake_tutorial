## 🚀 Launch the Tutorial (No Install Required)

This workshop uses Ona, an AI agent platform, to provide a pre-configured cloud environment. Click **Open in Ona** down below to start your personal workspace:

[Open in Ona](https://app.ona.com/#https://github.com/Dhananjhay/khanlab_snakemake_tutorial)

### ⏱️ What happens next?
1. **Sign in:** You'll be asked to log in with your GitHub account.
2. **Environment Build:** Ona will spend ~60 seconds setting up Python and Snakemake.
3. **Ready to Code:** A VS Code-like editor will open in your browser with the MRI data already loaded in `data/`.
